##📚 Kitoblar Market Sayti

Zamonaviy va responsive (moslashuvchan) kitoblar market sayti. Loyiha React va Vite yordamida yaratilgan bo‘lib, foydalanuvchilarga kitoblarni ko‘rish va interfeys orqali oson boshqarish imkonini beradi.

🚀 Loyiha haqida

Ushbu loyiha frontend asosida yaratilgan kitoblar marketi hisoblanadi. Asosiy e’tibor quyidagilarga qaratilgan:

Toza va zamonaviy dizayn

Responsive layout (telefon, planshet, kompyuter uchun mos)

Component-based arxitektura

Soddalashtirilgan va qulay foydalanuvchi interfeysi

Saytda Hero qismi, kitoblar ro‘yxati, CTA bo‘limi va Footer mavjud.

🛠️ Ishlatilgan texnologiyalar

React – Komponentlarga asoslangan UI

Vite – Tezkor build va development muhiti

Mantine UI – Tayyor zamonaviy komponentlar kutubxonasi

Tailwind CSS – Utility-first CSS framework

JavaScript (ES6+)

✨ Asosiy imkoniyatlar

📖 Kitoblar ro‘yxatini ko‘rish

🔄 Loader (yuklanish animatsiyasi)

🪟 Modal oynalar

📱 To‘liq responsive dizayn

🎨 Minimal va zamonaviy interfeys

⚡ Tez ishlash va optimallashtirilgan build

📂 Loyiha tuzilmasi

MainPage – Asosiy sahifa komponentlari

maincomponents – Qayta ishlatiladigan komponentlar

components/ui – UI kutubxonaga oid komponentlar

assets – Rasmlar va statik fayllar
